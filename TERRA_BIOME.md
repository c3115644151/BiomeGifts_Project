# Terra Biome Classification for BiomeGifts

## 1. 热带群系 (Tropical / Hot)
**特征**: 炎热、干燥或湿热。对应配置关键字: `DESERT`, `BADLANDS`, `SAVANNA`, `JUNGLE`, `TROPICAL`.

### Dry / Hot (干燥/炎热)
- **Desert (沙漠类)**
  - `DESERT`
  - `DESERT_FLATS`
  - `DESERT_PILLARS`
  - `DESERT_SPIKES`, `DESERT_SPIKES_GOLD`
  - `ARID_SPIKES`
  - `ARID_HIGHLANDS`
  - `ROCKY_DESERT`
  - `SALT_FLATS` (盐碱地)
- **Badlands (恶地类)**
  - `CRACKED_BADLANDS_PLATEAU`
  - `BADLANDS_BUTTES`, `ERODED_BADLANDS_BUTTES`
  - `BADLANDS_MOUNTAINS`, `BADLANDS_MOUNTAINS_RIVER`
  - `WOODED_BUTTES` (Temperate but hot?) -> Usually warm.
  - `TERRACOTTA_SEA_ARCHES`, `TERRACOTTA_SEA_CAVES`
- **Savanna (热带草原类)**
  - `SAVANNA`, `SAVANNA_LOW_HILLS`, `SAVANNA_HILLS`, `SAVANNA_OVERHANGS`
  - `GRASS_SAVANNA`, `GRASS_SAVANNA_LOW_HILLS`, `GRASS_SAVANNA_HILLS`
  - `OAK_SAVANNA`
  - `SHRUBLAND`, `DRY_SHRUBLAND`
  - `CHAPARRAL`, `LOW_CHAPARRAL`, `CHAPARRAL_FLATS`
  - `XERIC_LOW_HILLS`, `XERIC_HILLS`, `XERIC_PLAINS`, `XERIC_MOUNTAINS`

### Wet / Humid (湿热/雨林)
- **Jungle (丛林类)**
  - `JUNGLE`, `JUNGLE_HILLS`, `JUNGLE_FLATS`, `JUNGLE_MOUNTAINS`
  - `BAMBOO_JUNGLE`, `BAMBOO_JUNGLE_HILLS`, `BAMBOO_JUNGLE_FLATS`, `BAMBOO_JUNGLE_MOUNTAINS`
  - `RAINFOREST`, `RAINFOREST_HILLS`
  - `MONSOON_MOUNTAINS`, `LARGE_MONSOON_MOUNTAINS`
  - `BAMBOO_PONDS`
- **Other Tropical**
  - `TROPICAL_OCEAN`, `TROPICAL_DEEP_OCEAN`
  - `CORAL_OCEAN`
  - `PALM_BEACH`, `PALM_FOREST`
  - `SANDSTONE_ARCHIPELAGO`, `SANDSTONE_WETLANDS`

## 2. 温带群系 (Temperate / Mild)
**特征**: 气候适中，四季分明。对应配置关键字: `FOREST`, `PLAINS`, `MEADOW`, `SWAMP`.

### Forest / Plains (森林/平原)
- **Forest (森林类)**
  - `FOREST`, `FOREST_HILLS`, `FOREST_LOWLANDS`, `FOREST_FLATS`
  - `BIRCH_FOREST`, `BIRCH_FOREST_HILLS`, `BIRCH_FLATS`
  - `DARK_FOREST`, `DARK_FOREST_HILLS`
  - `EUCALYPTUS_FOREST`
  - `FLOWERING_FOREST`, `FLOWERING_FOREST_HILLS`, `FLOWERING_FLATS`
  - `AUTUMNAL_FOREST`, `AUTUMNAL_FOREST_HILLS`, `AUTUMNAL_FLATS`
  - `SAKURA_MOUNTAINS`
  - `EVERGREEN_FOREST`, `EVERGREEN_FOREST_HILLS`, `EVERGREEN_FLATS`, `EVERGREEN_OVERHANGS`
  - `XEROPHYTIC_FOREST`, `XEROPHYTIC_FOREST_HILLS`
- **Plains (平原类)**
  - `PLAINS`, `SUNFLOWER_PLAINS`
  - `PRAIRIE`, `STEPPE`
  - `HIGHLANDS`, `WILD_HIGHLANDS`, `DRY_WILD_HIGHLANDS`
  - `MOORLAND`
  - `PALE_GARDEN`
- **Mountains (山地类 - 温带)**
  - `TEMPERATE_MOUNTAINS`, `TEMPERATE_ALPHA_MOUNTAINS`
  - `DRY_TEMPERATE_MOUNTAINS`, `DRY_TEMPERATE_WHITE_MOUNTAINS`
  - `WILD_BUMPY_MOUNTAINS`, `ROCKY_BUMPY_MOUNTAINS`
  - `CERROS_DE_MAVECURE`

### Wet / Swamp (湿地/沼泽)
- **Swamp (沼泽类)**
  - `SWAMP`, `MANGROVE_SWAMP`
  - `WETLANDS`, `MARSH`
  - `ROCKY_WETLANDS`
  - `LUSH_SEA_CAVES`
- **Water (水域)**
  - `RIVER`
  - `OCEAN`, `DEEP_OCEAN`, `DEEP_OCEAN_TRENCH`
  - `SUBTROPICAL_OCEAN`, `SUBTROPICAL_DEEP_OCEAN`
  - `ARCHIPELAGO`, `TEMPERATE_SEA_ARCHES`

## 3. 寒带群系 (Cold / Snowy)
**特征**: 寒冷、冰雪覆盖。对应配置关键字: `SNOW`, `ICE`, `FROZEN`, `TAIGA`.

### Snow / Ice (冰雪)
- **Snowy (雪原类)**
  - `SNOWY_PLAINS`, `SNOWY_MEADOW`
  - `SNOWY_MOUNTAINS`, `SNOWY_TERRACED_MOUNTAINS`, `SNOWY_ERODED_TERRACED_MOUNTAINS`
  - `TUNDRA_PLAINS`, `TUNDRA_MIDLANDS`, `TUNDRA_HILLS`
- **Ice (冰川类)**
  - `ICE_SPIKES`
  - `FROZEN_BEACH`, `FROZEN_ARCHIPELAGO`, `FROZEN_WETLANDS`, `FROZEN_MARSH`
  - `SNOWY_SEA_ARCHES`, `SNOWY_SEA_CAVES`
  - `FROZEN_RIVER`, `FROZEN_OCEAN`, `FROZEN_DEEP_OCEAN`, `ICEBERG_OCEAN`
  - `COLD_OCEAN`, `COLD_DEEP_OCEAN`

### Taiga (针叶林)
- **Taiga (泰加林)**
  - `TAIGA`, `TAIGA_HILLS`, `TAIGA_FLATS`
  - `REDWOOD_FOREST_HILLS`
  - `MOUNTAINS` (Generic mountains usually cold/stony)
  - `ROCKY_ARCHIPELAGO`, `ROCKY_SEA_ARCHES`, `ROCKY_SEA_CAVES`

## 4. 特殊群系 (Special)
- **Mushroom (蘑菇岛)**: `MUSHROOM_COAST`, `MUSHROOM_FIELDS`, `MUSHROOM_HILLS`, `MUSHROOM_MOUNTAINS`
- **Volcanic (火山)**: `ACTIVE_VOLCANO_*`, `CALDERA_VOLCANO_*`, `YELLOWSTONE`
- **Cave (洞穴)**: `DRIPSTONE_CAVES`, `DEEP_DARK`, `LUSH_CAVES`
